About Inventory Management System

System Overview

This Inventory Management System is a comprehensive solution designed to help businesses track and manage their inventory efficiently. Built with modern web technologies, it provides real-time inventory tracking, supplier management, and detailed reporting capabilities.

Key Features
  • Real-time inventory tracking
  • Product and supplier management
  • Transaction history and audit trails
  • Low stock alerts and notifications
  • Comprehensive reporting system
  • User role management (Admin/Staff)
  • Responsive design for mobile and desktop
  • Export capabilities (CSV/PDF)
Version 1.0

Built with Flask, SQLAlchemy, and Bootstrap

Technology Stack
Backend:
  • Python Flask
  • SQLAlchemy ORM
  • SQLite Database
  • Werkzeug Security
Frontend:
  • HTML5 & CSS3
  • Bootstrap 5
  • JavaScript
  • Font Awesome Icons
System Requirements
  • Python: 3.7 or higher
  • Flask: 2.3.3
  • Database: SQLite (included)
  • Browser: Modern web browser with JavaScript enabled
Default Credentials
Admin: admin / admin123

Please change these passwords in production!
Getting Started
  1. Log in with your credentials
  2. Add suppliers to the system
  3. Create product entries with supplier information
  4. Record inventory transactions (stock in/out)
  5. Monitor stock levels and generate reports
Tip: Use the dashboard to get a quick overview of your inventory status, including low stock alerts and recent transactions.
